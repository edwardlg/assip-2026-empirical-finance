    Resources¶

    Everything you need to do the work — separate from the operating schedule and the team pages.

    •   Setup

      Conda env (Python 3.11), VS Code, Quarto, MkDocs, GitHub auth. Start here on day one of orientation.

    •   Data access

      WRDS (via GMU institutional seat), FRED API key, HMDA via the CFPB Data Browser, GitHub repo setup, env-var discipline, seed pinning.

    •   Reading list

      The ten anchor papers — Prof. Gao's own work plus the empirical-finance classics — with JSTOR/SSRN links and a one-sentence "why it matters" for each.

    •   Data cards

      Every dataset we touch has a one-page data card in the textbook at data-cards/. Browse them in the textbook.

    •   Cheat sheets

      pandas, NumPy, statsmodels, pyfixest, matplotlib quick-reference (one page each). Coming in Week 2.

    •   The Quarto textbook

      The primary text lives at the sibling Quarto site. The weekly pages on this wiki link directly into the relevant chapters and notebooks.

    A note on the boundary between this wiki and the textbook¶

    • The wiki (this site) is the operating dashboard — schedule, deliverables, cohort, projects. It changes weekly.
    • The textbook (Quarto site) is the content — chapters, problem sets, lab notebooks, data cards, solutions. It is largely stable once shipped.

    If something on the wiki contradicts something in the textbook about what to read or what to submit, the wiki wins — it is the live one.

    If something on the wiki contradicts the textbook about the math or the code, the textbook wins — open an issue and we will fix the wiki.
